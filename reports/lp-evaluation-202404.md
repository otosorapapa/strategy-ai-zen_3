## 3. 主要改善提案（ユーザー心理×コンバージョン理論）

| 改善提案 | 狙い・根拠 |
| --- | --- |
| ファーストビューに「3つの即価値」サマリーカードを追加し、本文を要約＋詳細の二段階構成に圧縮 | AIDMAのAttention→Interestで初期理解を高速化し、長文読解による離脱を防止。スクロール前に価値を把握させる。 |
| フォームをステップ化し（①メール＋希望内容 → ②任意詳細）、送信ボタン直下に完了後フローをビジュアル化 | AIDMAのAction段階で入力摩擦を低減。段階分割と「送信後に得られるメリット」提示で完遂率を引き上げる。 |
| CTA直前に金融機関コメントと成果指標を再掲 | PASのSolution→Actionで意思決定直前の不安を解消。成果実例を再提示し確信を補強。 |
| H2/H3に地域・課題系キーワードを追記し、FAQで検索想起語を網羅 | AISASのSearch段階を強化。ロングテール流入の意図一致を高める。 |

## 4. 改善後の想定効果（定量推定）

| 施策 | 想定指標変化 |

| フォームステップ化＋送信後フロー可視化 | フォーム完了率 +18〜25%、CTAクリック後離脱率 ▲20〜30%。入力心理負担を軽減。 |

| 見出しキーワード最適化 | 指名外有機流入 +8〜12%、セッションあたり滞在時間 +10〜14%。検索意図との一致向上。 |
